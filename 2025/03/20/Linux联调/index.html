配置IDA远程调试

其中，填入信息如下：

选项 应填入的信息
Application 待调试文件在Linux上的路径（包括待调试文件）
Director 调试文件所在的文件夹路径
Parameters 运行参数，例如运行可执行文件时传递到main函数的命令参数
Hostname Linux的IP地址
Port Linux的端口，这里默认即可，和Linux上运行的linux_serverx64开启的端口一致
Password Linux登陆密码
